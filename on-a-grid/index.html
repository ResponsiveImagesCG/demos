A responsive image, a fluid grid and the <picture>-element

This is a test page for the picture element prototype. The Demo has been built on top of a fluid Grid with 5 Breakpoint as mentioned below. The aim of this document is to show the benefit of an individual image resource for each of this breakpoints. The demo follows a Mobile First approach!

Author: Andreas Klein / @ixisio

Instructions to see the magic in action

  1. Load the page small browser window.
  2. Now increase the size of your UA window
  3. You will see how ugly the image appears.
  4. Reload the page
  5. You now will see an other (breakpoint optimized) image.

Resolutions / Breakpoints

  1. @mobile: 320px * 192px (7kb)
  2. @wide-mobile: 480px * 288px (17kb)
  3. @tablet: 768px * 514px (41kb)
  4. @desktop: 1024px * 686px (90kb)
  5. @highres: 1280px * 852px (253kb)

A very exciting yacht race.

The source code:

                
<!-- start: picture-element -->
<picture width="100%" height="100%">

    <!-- start: Handheld-Devices -->
    <!-- Source intended for: Mobile Layout -->
    <source media="(max-width: 30em)" src="img/yacht_race@mobile.jpg">
    <!-- Source intended for: Wide Mobile Layout -->
    <source media="(min-width: 30em) and (max-width: 48em)" src="img/yacht_race@wide-mobile.jpg">
    <!-- Source intended for: Tablet Layout -->
    <source media="(min-width: 48em) and (max-width: 60em)" src="img/yacht_race@tablet.jpg">
    <!-- end: Handheld Devices -->

    <!-- start: Stationary Devices -->
    <!-- Source intended for: Notebooks and normal Desktop Layouts -->
    <source media="(min-width: 60em) and (max-width: 80em)" src="img/yacht_race@desktop.jpg">
    <!-- Source intended for: Notebooks and normal Desktop Layouts -->
    <source media="(min-width: 80em)" src="img/yacht_race@hires.jpg">
    <!-- end: Stationary Devices -->
    
    <!-- start: fallback img-element if picture-element is not supported -->
    <img src="img/yacht_race@fallback.jpg" alt="A very exciting yacht race.">
    <!-- end: fallback -->

    <!-- start: alternate text -->
    <p>A very exciting yacht race.</p>
    <!-- end: alternate text -->

</picture>
<!-- end: picture-element -->
