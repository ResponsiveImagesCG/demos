<picture>-element demo

This is a test page for the picture element prototype.

Author: Brian Muenzenmeyer / @bmuenzenmeyer

Instructions:

  1. Load the page in a browser window
  2. You will see an image of a hamburger.
  3. Print the page.
  4. You will see a coupon of the hamburger special with details about the special.

Meat Eater Monster Sandwich Special

The source code:

                
<!-- picture-element -->
<picture>
    <source media="screen and (max-width: 479px)" src="img/monster_small.jpg">
    <source media="screen and (min-width: 480px)" src="img/monster.jpg">
    <source media="print" src="img/monster_print.jpg">

    <!-- fallback img-element if picture-element is not supported -->
    <img src="img/monster.jpg">
                
    <!-- a11y: providing alternate text -->
    <p>Meat Eater Monster Sandwich Special</p>
</picture>
