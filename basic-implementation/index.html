<picture>-element demo

This is a test page for the picture element prototype.

Author: Anselm Hannemann / @anselmhannemann

Instructions:

  1. Load the page large browser window
  2. On large window you will see the whole castle-set with lake and sky.
  3. Now narrow your UA window
  4. Reload the page
  5. You now will see an other crop of the image: just the main part of the castle.

Resolutions:

  1. 240px * 160px
  2. 480px * 320px
  3. 960px * 640px
  4. monochrome

Nymphenburg Castle in Munich during sunset

The source code:

                
<!-- picture-element -->
<picture>
    <source media="(max-width: 479px)" src="test_landscape_1@1x.jpg">
    <source media="(min-width: 480px) and (max-width: 639px)" src="test_landscape_1@2x.jpg">
    <source media="(min-width: 640px)" src="test_landscape_1@4x.jpg">
    <source media="monochrome" src="test_landscape_1@monochrome.jpg">
    <source media="print" src="test_landscape_1@monochrome.jpg">

    <!-- fallback img-element if picture-element is not supported -->
    <img src="test_landscape_1@2x.jpg">
                
    <!-- a11y: providing alternate text -->
    <p>Nymphenburg Castle in Munich during sunset</p>
</picture>
